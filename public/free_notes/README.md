# How to Add Your PDFs

## FREE Notes (Public Access)

Add your PDF files to this folder: `public/free_notes/`

### Files to Upload:
- bhdla-135.pdf (Hindi Sahitya ka Itihas)
- bevae-181.pdf (Environmental Studies)

### After Upload:
They will be accessible at:
- https://your-vercel-url.vercel.app/free_notes/bhdla-135.pdf
- https://your-vercel-url.vercel.app/free_notes/bevae-181.pdf

### Steps:
1. Replace the .txt placeholder files with your actual PDF files
2. Make sure filenames match exactly
3. Run: `git add public/`
4. Run: `git commit -m "Add PDF notes"`
5. Run: `git push`
6. Vercel will auto-deploy!

---

## PAID Notes (For Future)

When you want to sell paid notes, create another folder:
- `public/paid_notes/`

Then add your paid PDFs there.

---

## Important:
- File names MUST match the product IDs in your website
- Use lowercase and hyphens
- Keep files under 10MB for best performance
